Security

All note files is stored in local folder. Without Marboo, accessing the content is available. All Marboo does is responsing your operations with folder.

Syncing with your pleasure

Supporting using Dropbox or Git to sync your notes to everywhere.

Custom Note UI is available

Modify the main CSS file to custom the UI of all notes.

Sharing notes in local network

Tell your friends the note link, the one in the same network can view your note.(Close Sharing? Just restart Marboo)

Note, Blog，All in one

Using Marboo to manage files in Jekyll / Octopress / Hexo / Ghost is a smart choice. Notes, Blogs, they are the same. Post to WordPress / Ghost is available!

Mananging Codes for Coders

Pygments inside. All kinds of code files can display with syntax highlight.
